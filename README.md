# Accurate_Localization_of_Static_Objects_Using_a_Dynamically_Moving_3D_Camera-Manipulator

# 개요
- 본 프로젝트는 카메라와 매니퓰레이터가 **동적으로 움직이는 상황에서도  정적 물체의 3차원 위치를 정확하게 추정하는 문제**를 다룬다.
- 로봇 주행 또는 매니퓰레이터 움직임으로 인해 카메라 포즈가 지속적으로 변하는 환경에서,  TF 체인(Camera → Base → World)을 활용하여  물체의 좌표를 월드 좌표계 기준으로 일관되게 추적.
- 이를 통해 단일 프레임 인식의 한계를 극복하고,  여러 시점에서 관측된 정보를 통합하여 **보다 안정적인 물체 위치 추정**을 수행.  본 단계는 실제 로봇 환경에서 요구되는 **동적 인식 문제를 해결하는 핵심 단계**.
